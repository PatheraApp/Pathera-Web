# Pathera Web

Marketing site for **pather.net** — separate from the mobile app (`liora`).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) → redirects to `/tr`.

## Locales

- `/tr` — Turkish
- `/en` — English
- `/ru` — Russian

## Theme

- **Dark** — reference mockup (gold accents)
- **Light** — mobile Pathera palette (olive, cream, violet)

Toggle with the sun/moon button in the header.
